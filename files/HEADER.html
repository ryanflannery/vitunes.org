Note the Following (especially for porters):
  • Files named vitunes-0.9* are ancient releases, and should be avoided.
  • Files named vitunes-p1.0.* are pre-1.0 releases, and should also be avoided.
  • Modern snapshots, which should be preferred, follow the usual naming/versioning scheme of vitunes-X.Y.Z.tgz.
    • Changes in Z alone denote small changes that (should) be OK to update to, as-is.
    • Changes in Y denote larger changes that would probably require heavier porting efforts.
    • Don't expect X to change for a while. ;D
